# 🎮 Game's Cord

> 게임 메이트를 매칭하고 코인으로 결제하는 게임 파티 매칭 웹 서비스

<br>

## 프로젝트 개요

리그오브레전드, 오버워치 등 파티 게임에서 함께 플레이할 게임 메이트를 구하고,  
코인 결제 시스템을 통해 유료로 매칭 서비스를 제공하는 웹 애플리케이션입니다.

| 항목 | 내용 |
|------|------|
| 진행 기간 | 2025 |
| 팀 구성 | 7인 팀 프로젝트 |
| GitHub | [game-s-cord](https://github.com/SB135246/game-s-cord) |

<br>

## 기술 스택

| 분류 | 기술 |
|------|------|
| **Backend** | Java, Spring Boot |
| **Frontend** | JavaScript, CSS, HTML, React |
| **Database** | MySQL |
| **협업** | Git, GitHub |

<br>

## 주요 기능

| 기능 | 설명 |
|------|------|
| 👤 회원 관리 | 회원가입 / 로그인 / 로그아웃 / 마이페이지 정보 수정 |
| 🎮 게임 메이트 등록 | 티어, 요금, 이용 가능 시간 등 메이트 프로필 등록 |
| 🔍 메이트 검색 | 게임 종류 / 티어 / 성별 / 평점순 등 다중 필터링 검색 |
| 💳 결제 및 코인 시스템 | 카드·계좌이체·간편결제 기반 코인 충전, 24시간 내 환불 처리 |
| 📋 매칭 관리 | 매칭 요청 / 수락 / 거절 / 취소 기능 |
| ⭐ 리뷰 시스템 | 매칭 완료 후 평점 및 리뷰 작성, 답변, 신고 기능 |
| 🔔 알림 시스템 | 매칭 요청·수락·거절 시 실시간 브라우저 알림 |
| ❤️ 즐겨찾기 | 메이트 즐겨찾기 추가 및 해제 |

<br>

## 👨‍💻 담당 역할 (서상범)

- **OAuth2 소셜 로그인 구현** — Google 소셜 로그인 인증 흐름 설계 및 구현, 소셜 계정의 비밀번호 재설정 요청 차단 처리
- **게임메이트 API 개발** — 게임메이트 조회 / 등록 / 수정 REST API 구현, 이용 가능 시간(start/end) 정보 처리
- **초기 환경 설정** — Spring Boot Config 설정, 더미데이터 구성, 클래스 다이어그램 작성

<br>
